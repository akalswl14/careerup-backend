# JobUp-Backend

JobUp의 API 레포지토리 입니다

## Installation

```bash

$ npm install

```

## Running the app

```bash

# development

$ npm run start



# watch mode

$ npm run start:dev



# production mode

$ npm run start:prod

```

## Test

아직 테스트는 구현되지 않았어요!

```bash

# unit tests

$ npm run test



# e2e tests

$ npm run test:e2e



# test coverage

$ npm run test:cov

```

## Database Setting

- 라비앙팀 노션을 참고해주세요
